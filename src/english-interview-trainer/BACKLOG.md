# Project Backlog

## Kids Mode English Tutor Adaptation
**Status:** Planned / Deferred

**Goal:** Transform the AI Interview Trainer into a playful, engaging, and gamified English learning tool tailored for children.

### Architecture Points:
1. **Dedicated Route (`/kids-tutor`)**: Separate from the professional UI to allow vibrant colors and gamified elements.
2. **Gamified Metrics**: Replace "Confidence %" with a Star System (⭐⭐⭐) and an XP progress bar.
3. **Tailored Setup Wizard**: Name, Age, Interests (Dinosaurs, Space, Animals).
4. **Kids-Specific AI Prompt**: A fun "English Buddy" persona. A1/A2 vocabulary. Use of `[FUN FEEDBACK]`, `[HOW TO SAY IT]`, `[NEXT ADVENTURE]`. Abundant emojis.
